다중 파일 처리(같은 종류의 여러 개의 파일 처리) 기능 구현 완료
UI 구성 완료
코드 main_ui.py에서 모듈을 import하여 실행하도록 구성했고, pyinstaller로 만든 프로그램 파일을 메일로 첨부해드렸습니다.
